= Open City site template

Simple site template using Ruby, Sinatra and Haml with tie-ins to Heroku and Google Analytics.

== Installation

  git clone git@github.com:open-city/site_template.git
  cd site_template
  gem install bundler
  bundle
  unicorn
  navigate to http://localhost:8080/

== Dependencies

  Ruby
  Sinatra
  Haml
  Heroku
  Google Analytics
